# 🚀 Asteroids (Python)

A simple recreation of the classic **Asteroids** arcade game built in Python. Navigate your spaceship through space, avoid incoming asteroids, and shoot them down to survive as long as possible.

---

## 🧠 About This Project

This is a **personal project** that I built over the course of **2 days** to practice programming in Python and strengthen my understanding of game development concepts.

The project focuses on:
- Object-oriented programming
- Game loops and real-time input handling
- Basic collision detection and movement physics
- Working with a game development library (Pygame)

---

## 🎮 Gameplay Overview

- Control a spaceship in open space
- Rotate and move in all directions
- Shoot bullets to destroy asteroids
- Avoid collisions — getting hit ends the game
- Survive as long as possible and improve your score

---

## ⚙️ Requirements

Before running the game, make sure you have the following installed:

- **Python 3.x**
- **pip** (Python package manager)
